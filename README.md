# Hello Kitty Adventure Game

This is a 2D adventure game originally built in Java and later extended into an Android application using Android Studio with Kotlin/Gradle integration.

## Features
- Player movement and navigation
- Item collection
- Interactive environments
- Android mobile support with on-screen controls

## Tech Stack
- Java
- Kotlin (Android integration)
- Android Studio / Gradle
- Git

## Project Structure
- `java-version/` – original desktop Java version  
- `android-version/` – Android Studio mobile version  

## About
This project was created as part of my coursework and expanded to explore mobile development and cross-platform integration.
